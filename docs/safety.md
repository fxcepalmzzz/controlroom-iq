# Safety and Synthetic Data Statement

ControlRoom IQ uses synthetic data only.

The project does not include:
- real employee data
- real customer data
- personally identifiable information
- credentials
- confidential company policies
- autonomous execution against real systems

The system is designed as a training simulator. It does not make real HR, procurement, support, compliance, or security decisions.

The core safety principle is human-in-the-loop supervision: learners practise when to approve, reject, ask for evidence, escalate, or pause AI recommendations.
